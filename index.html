git basics

initializing repository 

$ cd /directory
$ git init

clone repository 

$ cd /directory
$ git clone https://github.com/nanodotbot/_git-short _git-short

getting list of remotes 

$ git remote
$ git remote -v

check status of staging area 

$ git status

adding files to the staging area 

$ git add filename
$ git add .

checking differences between working directory and staging area 

$ git diff filename
$ git diff

storing changes from staging area inside the repository 

$ git commit filename -m 'just some changes'
$ git commit . -m 'all my current changes'

view commit history 

$ git log

push repository 

$ git push -u origin master

get changes from remote into local repository onto a remote branch 

$ git fetch

$ git fetch master
$ git fetch --all

integration origin/master into local master 

$ git merge origin/master
